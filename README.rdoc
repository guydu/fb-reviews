== Facebook Ads generator

This tool utlizes un-used page data to create content and personalized ads.

Each page review can be transformed easily to facebook image ad.

The tool uses the following interfaces

* Facebook graph API

* Facebook ads management API

* Cloudinary

* Facebook Scraping :/

Tool's flow is the following:

* Login

* Refresh to get up to date reviews

* clicking "Create Ad" will create image ad for that specific review
